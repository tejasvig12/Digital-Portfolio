Two purposes of a secure HTTPS connection is that it secures communication and the data transfer between the user and the website. It also encrypts all the message contents. An HTTPS Proxy Appliance is something that secures a web server and have a "gurantee of Internet browser privacy and security"(Gibson 2020 p1). An MITM is short for "man in the middle attack" which is a cyberattack where communication between two parties is intercepted often to steal personal information and corrupt data. This presents many risks because if someone stole your data such as your credit card number they could use that number and sell it to other third party dealers which would mean a loss of money. A hash is a process that can be used to see if various types of input or valid and authentic. Qualities that make a hash "good" are that it should uniformly distribute the keys and it should be efficiently computable. A CA is short for Certificate Authority, and the role that it plays is that it validates the identities of websites and issues electronic documents which are known as digital certificates. SSL interception can be prevented by not sharing sensitive information on non HTTPS sites, avoiding clicking on spam emails, and always using a safe computer when browsing the Internet. SSL inteception can be detected if your laptop is monitored by an organization that you're a part of, such as this laptop being monitored by Allen ISD. A false positive is when a test result shows the wrong result of a condition and a false negative is the opposite error where the test result incorrectly shows that the conditions is not there when it is actually present.
